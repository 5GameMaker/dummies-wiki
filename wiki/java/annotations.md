# Annotations

## Processors

All annotation processors must implement `javax.annotation.processing.Processor`[^1]. Although usually
`javax.annotation.processing.AbstractProcessor` is extended.



[^1]: https://docs.oracle.com/javase/8/docs/api/javax/annotation/processing/Processor.html
[^2]: https://stackoverflow.com/questions/11685498/what-is-the-default-annotation-processors-discovery-process
