# Code of Conduct

This document outlines the rules which you must follow when you interact with
this repository and surrounding communities.

1. Be nice
> Don't attack others for no reason, thanks.
>
> Maybe tune down the rudeness too.

2. Don't bring in drama
> Keep that on Twitter, thanks!

3. Gay is good
> No homophobia, transphobia, etc. 

Violations will get you banned from the project.
